Simulador de Evaluación de Contenido para Plataforma de Streaming

Este proyecto es un programa desarrollado en C# que simula el proceso de evaluación de contenido en una plataforma de streaming.

El sistema permite analizar distintos contenidos (películas, series, documentales, eventos en vivo) y tomar una decisión automática basada en reglas definidas.

Tiene como objetivo:
Evaluar contenido ingresado por el usuario y determinar:
- Ser publicado
- Enviado a revisión
- Ser rechazado

Su funcionalidad:
1.) Evaluar nuevo contenido
Permite ingresar los datos del contenido:

- Tipo de contenido
- Duración
- Clasificación
- Hora de transmisión
- Nivel de producción

El sistema procesa la información y muestra:

Impacto (Alto, Medio o Bajo)
Decisión final (Publicar, En revisión o Rechazar)

2.) Mostrar reglas del sistema
Muestra las reglas utilizadas para evaluar el contenido de forma clara y sencilla para el usuario.


3.) Mostrar estadísticas

- Total de contenidos evaluados
- Cantidad de publicados
- Cantidad de rechazados
- Cantidad en revisión
- Impacto predominante
- Porcentaje de aprobación

4.) Reiniciar estadísticas

Permite reiniciar todos los contadores del sistema a cero.

S5.) Salir
Finaliza el programa mostrando un resumen final de estadísticas.

Uso del programa
Ejecutar el programa
Seleccionar una opción del menú
Ingresar los datos solicitados
Visualizar el resultado en pantalla
Presionar una tecla para continuar

El sistema está diseñado para ser:

Fácil de usar
Claro para cualquier usuario
Resistente a errores de entrada

Simula de forma sencilla cómo una plataforma puede tomar decisiones automáticas sobre contenido.
